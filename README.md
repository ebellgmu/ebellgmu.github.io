# ebellgmu.github.io
This GitHub repository contains an interactive narrative created in Twine using HTML format.  The interactive Twine story revolves around the life of Anne of Cleves during her marriage to King Henry VIII in Tudor England.  It utilizes counterfactual history to engage the player while providing educational information about Tudor history.
